# 🚀 AI-Powered Code Reviewer

An intelligent full-stack web application that analyzes code snippets using AI and provides real-time feedback, suggestions, and improvements to enhance code quality and developer productivity.

---

## 🌟 Features

* 🔍 AI-based code analysis and review
* ⚡ Real-time suggestions for optimization and bug fixing
* 🧠 Supports multiple programming languages
* 💡 Clean and interactive user interface
* 🔐 Secure API integration
* 🚀 Fast and scalable MERN architecture

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Powered-Code-Reviewer.git
cd AI-Powered-Code-Reviewer
```

### 2️⃣ Install dependencies

#### Backend

```bash
cd backend
npm install --legacy-peer-deps
```

#### Frontend

```bash
cd frontend
npm install --legacy-peer-deps
```

---

### 3️⃣ Setup Environment Variables

Create a `.env` file in backend folder and add:

```
GEMINI_API_KEY=your_api_key_here
PORT=5000
```

---

### 4️⃣ Run the project

#### Backend

```bash
npm start
```

#### Frontend

```bash
npm run dev
```
